Instructions

Welcome to the ChatGPT Website! This website allows you to interact with ChatGPT, an AI assistant powered by OpenAI's GPT-4.

How to use the site

  1. Enter your API key in the index.html file in place of < API KEY >
  2. Enter a system message in the "System Message" text box and click "Send System Message". This message helps set the behavior of the AI assistant.
  3. Type your message or question in the "Type your message here..." text box.
  4. Press "Send" or hit the Enter key to submit your message.
  5. The AI assistant will respond in the output box above the input box.
  6. Continue the conversation by typing more messages and submitting them.
  7. If you want to save the conversation, click the "Save Session" button. This will download a text file containing the conversation.
  8. To clear the conversation and start a new one, click the "Clear Conversation" button.
